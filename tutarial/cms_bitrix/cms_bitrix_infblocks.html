CMS Bitrix --- инфоблоки, карточки товара и прочее

Инфоблок использует шаблон для вывода нужной информации на сайт, шаблон создается через "Типы информационных блоков"

    В нем возможно задать для будущей новости(товара, справочника, ) различные поля и свойства
  • URL для анонса и расширеного описания составленого из ID или символьных кодов
  • Для SEO продвижения назначить общие фразы и заголовки для всех новостей блока, раздела или всех страниц через подстановку шаблона
  • Определить что будет выводиться у каждой новости(товара), данные типа времени, цены или ....
  • Опредилить доступ просмотра, редактирования к контенту созданого инфоблоком

Создание инфоблока начинается с перехода по вкладке контент - типы инфоблоков - добавить новый тип

Назначаем ID блоку , именуем русское и ангглоязычное название типа инфоблока.

Создаем и именуем сам блок с новостями или каталогом

Создаем новость или товар в элементе инфоблока
