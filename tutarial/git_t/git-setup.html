Git --- Распределённая система контроля версий

Настройка Git , и файлы настройки

Файл .gitconfig

            [user]
            name = Michail
            email = verstka@mail.ru
            [color]
            ui = true
            [core]
            autocrlf = true
            safecrlf = true
            editor = nano

Прописываем пользователя репозитория

  • git config --global user.name "Michail"
  • git config --global user.email "verstka@mail.ru"

Цветовая гамма кода в консоли

  • git config --global color.ui true

Автоматический перенос строк

  • git config --global core.autocrlf input (#Linux/Mac)
  • git config --global core.autocrlf true (#Windows)
  • git config --global core.safecrlf true
