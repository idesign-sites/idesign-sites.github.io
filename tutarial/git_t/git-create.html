Создание репозитория в Git

Репозиторий это проект который хранится на диске(в папке)

Создать в папке проекта репозиторий можно командой 

                     git init

Создать в конкретной папке можно командой

                   git init Project

-------------------------------------------------- 

                    $ cd project
                    $ git init
                    Initialized empty Git repository in /home/michail/project/.git/

В папке проекта создастся файл .git в которой будет храниться файлы настройки проекта (config)

  
                  [core]
                        repositoryformatversion = 0
                        filemode = false
                        bare = false
                        logallrefupdates = true

filemode режим слежения за файлами установить в false

--------------------------------------------------

Проверка статуса проекта в Git

                     $ git status
                
         
                    # On branch master
                    #
                    # Initial commit
                    #
                    nothing to commit (create/copy files and use "git add" to track)

Изначально новый проект всегда создаётся в ветке MASTER эта команда в данном случае показала что папка репозитория пуста и файлов там нет

--------------------------------------------------

Создание файлов из консоли TOUCH

                     touch index.html
