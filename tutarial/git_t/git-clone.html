Клонирование удаленного репозитория в папку пк

В папке проекта скачаются файлы проекта и .git в которой будет храниться файлы настройки проекта (config)

  
                  $ git clone https://github.com/idesign-sites/idesign-sites.github.io.git
                    Cloning into 'idesign-sites.github.io'...
                    remote: Counting objects: 1000, done.
                    remote: Compressing objects: 100% (30/30), done.
                    remote: Total 1000 (delta 15), reused 0 (delta 0), pack-reused 970
                    Receiving objects: 100% (1000/1000), 44.67 MiB | 28.00 KiB/s, done.
                    Resolving deltas: 100% (251/251), done.
                    Checking out files: 100% (824/824), done.

В файл можно перепрописать параметры конфига для текущего рпозитория , например настройки логина и почты от рабочего репозитория
