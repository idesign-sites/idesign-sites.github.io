Горизонтальное расположение и выравнивание элементов в HTML документе.

Строчные HTML элементы

Расположение таких элементов в документе (в нормальном потоке) возможно при применении следующих правил CSS:

  • text-align; (left, right, center); значение устанавливается для родительского элемента(если строчный элемент не является анонимным)
    1. <p class = "outer_block" >
    2.   <span class = "inner_block" > строчный элемент </span>
    3. </p>
    1. .outer_block {

      text-align: center;

      }

    строчный элемент

  • padding: left(right);
  • margin: left(right);

Применение правил CSS изменяющих нормальный поток элементов:

  • position: (absolute, relative);
  • float: right(left);
